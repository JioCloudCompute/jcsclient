
NAME :
    delete-snapshot


DESCRIPTION :
    Delete Snapshot that are currently visible under your JCS account.


SYNOPSIS :
    delete-snapshot
    --snapshot-id <value>


OPTIONS :
    --snapshot-id
    snapshot-id that needs to be deleted.


EXAMPLES :
    jcs compute delete-snapshot --snapshot-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf

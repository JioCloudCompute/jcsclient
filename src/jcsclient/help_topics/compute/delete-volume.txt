
NAME :
    delete-volume


DESCRIPTION :
    Delete Volume that are currently visible under your JCS account.


SYNOPSIS :
    delete-volume
    --volume-id <value>


OPTIONS :
    --volume-id
    volume-id that needs to be deleted.


EXAMPLES :
    jcs compute delete-volume --volume-id b2ba5a18-d8cc-4223-bf62-2a866f5890bf
